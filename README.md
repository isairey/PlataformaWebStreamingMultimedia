# 🌌 Acerca del proyecto

**Movie Streaming Website** es una plataforma web diseñada para reproducir videos y películas directamente desde el navegador.

El sistema ofrece una experiencia multimedia moderna con:

- 🎬 Streaming de películas
- 📺 Reproducción de videos
- 🗄️ Gestión mediante base de datos
- 📱 Diseño responsive
- ⚡ Navegación rápida
- 🌐 Hosting local o remoto

El proyecto fue desarrollado para practicar:

- Desarrollo web con PHP
- Bases de datos MySQL
- Plataformas multimedia
- Arquitectura cliente-servidor
- Streaming web

# ⚡ Instalación

## 📋 Requisitos

- PHP
- MySQL
- XAMPP o WAMP

---

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

### SSH

```bash
git clone git@github.com:MDAffanMafia/movie-streaming-website.git
```

---

### HTTPS

```bash
git clone https://github.com/MDAffanMafia/movie-streaming-website.git
```

---

## 2️⃣ Configurar base de datos

Importar archivo:

```bash
db.sql
```

en MySQL o phpMyAdmin.

---

## 3️⃣ Ejecutar servidor

### Opción A — XAMPP / WAMP

- Colocar proyecto en `htdocs`
- Iniciar Apache y MySQL

---

### Opción B — PHP Internal Server

```bash
sudo php -S localhost:8000
```

⚡ Puedes usar cualquier puerto.
